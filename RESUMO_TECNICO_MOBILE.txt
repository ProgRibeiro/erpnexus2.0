ROTAS CONFIGURADAS

/tecnico-mobile
  - Lista de ordens do dia
  - Refresh automático
  - Cache offline

  - Captura de fotos

  - Chat
  - Sincronização

- hora_conclusao (timestamp)
- laudo (texto)
- assinatura_cliente (base64)

================================================================================
COMO USAR
================================================================================

1. Para o técnico:
   - Acesse /tecnico-mobile em dispositivo mobile
   - Veja suas ordens do dia
   - Clique em "Iniciar Atendimento"
   - Registre chegada
   - Tire fotos (antes e depois)
   - Preencha laudo
   - Obtenha assinatura
   - Finalize o serviço

2. Para desenvolvimento:
   - npm run dev
   - Abra http://localhost:5173/tecnico-mobile
   - Teste no Chrome DevTools (F12)
   - Simule offline em Network tab
   - Verifique IndexedDB em Application

3. Para produção:
   - npm run build
   - Faça deploy (Vercel, Netlify, Docker)
   - HTTPS obrigatório (câmera requer)
   - Monitore sincronizações
