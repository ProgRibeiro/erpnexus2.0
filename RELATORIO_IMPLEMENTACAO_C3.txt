🔧 COMO COMEÇAR

1. CONFIGURAR AMBIENTE (.env)
   └─ Consultar: erp_backend/.env.example

2. EXECUTAR MIGRAÇÕES
   └─ python manage.py migrate

3. INICIALIZAR TEMPLATES
   └─ python init_notificacoes.py

4. INICIAR CELERY
   ├─ Terminal 1: celery -A config beat -l info
   └─ Terminal 2: celery -A config worker -l info

5. TESTAR
   ├─ Admin: http://localhost:8000/admin/notificacoes/
   └─ API: POST /api/v1/notificacoes/logs/testar_notificacao/
